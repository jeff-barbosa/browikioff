O termo "GM" é a abreviação de Game Master, o "Mestre do Jogo", uma espécie de oficial da lei que existe para garantir o bom andamento do jogo, e deve ser considerado e respeitado como uma autoridade.Em Ragnarök, você poderá encontrar as denominações GM, GE e LU!. Todos eles são treinados para exercer a função e seguem regras específicas.

A equipe de GM's possui diversas tarefas, como por exemplo:

  • Checar a funcionalidade dos itens no servidor de testes;
  • Verificar possíveis anormalidades no jogo;
  • Fazer com que as Regras de Conduta sejam cumpridas;
  • Aplicar punições aos jogadores infratores.

Identificação

É fácil identificar um Game Master em Ragnarök, basta atentar-se para a roupa exclusiva que esses personagens usam, além de uma série de fatores:

  • Os nomes e mensagens públicas dos GM's são exibidas em amarelo.
  • GM's nunca pedem detalhes da conta, como e-mail, senha e nome de usuário.
  • GM's nunca pedirão itens ou zenys.
  • GM's nunca enviam mensagens particulares alertando uma suspensão da conta ou punição.
  • Quando você receber uma mensagem particular de um Game Master, ela aparecerá também via broadcast somente para você. Isso significa que se você receber uma mensagem particular de alguém se intitulando GM e não aparecer um Broadcast, essa pessoa é uma fraude.
  • Game Masters possuem identificação padronizada pela empresa. Antes, eles possuíam nomes, mas recentemente, foram alterados:
    • Antes: [GM]Kouta
    • Agora: [GM]bROx
      • onde x representa um número entre 1 a 100.
  • Observações importantes:
    • Desde o dia 10.05.2011, os GM's deixaram de atender os jogadores in-game para atendê-los através dos tickets da Central de Suporte.
    • Se você encontrar algum personagem se fazendo passar de Game Master ou amigo de GM, você deverá contactar imediatamente a Central de Suporte.

Trabalhando

GM's trabalham em regime de escala, isso significa que seus horários são divididos entre os turnos da manhã, tarde e noite.

Para tornar-se um Game Master, o jogador precisa contemplar as seguintes requisições:

  1. Ter o ensino médio completo;
  2. Ser maior de idade e residir em São Paulo;
  3. Possuir conhecimentos de Windows e aplicações Office (Excel, Word,...);
  4. Possuir familiaridade com jogos eletrônicos, em especial com jogos online;
  • Envie seu currículo.

Comandos

Para facilitar o trabalho da equipe de monitoramento, os Game Masters possuem comandos exclusivos. É interessante conhecê-los para que não existam dúvidas quanto à performance de um GM.

Se você encontrar alguém se passando por GM, peça para que ele use o comando /summon em você, como prova.


  • /b <mensagem>: Envia um Broadcast com o nome para todos no servidor.
  • /nb <mensagem>: Envia um Broadcast sem nome para todos no servidor.
  • /bb <mensagem>: Envia um Broadcast de cor azul para todos do mapa.
  • /nlb <mensagem>: Envia um Broadcast somente para todos do mapa.
  • /mm <mapa>: Teleporta o GM para o mapa descrito.
  • /shift <nome>: Move o GM diretamente para um jogador.
  • /remove: Move o GM para a conta do jogador, em que personagem ele estiver.
  • /recall <nome>: Teleporta um jogador pelo nome de usuário da conta.
  • /summon <nome>: Teleporta um jogador para o GM.
  • /hide: Torna do Game Master invisível para todos, não pode ser detectado por habilidades como Revelação nem ser atacado por Monstros. Ainda é possível andar e utilizar habilidades como Santuário.
