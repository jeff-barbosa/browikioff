Essa predefinição serve para padronizar todas páginas de Quests inseridas na Wiki. Copie e cole o código abaixo para criar a predefinição.


Uso

{{Quest Info| levelreq =| joblevelreq =| classreq = | classreq2 = | skillreq =| partyreq = | itemreq =| equipreq = | zenyreq = | hunting =| questsreq =| baseexpreward =| jobexpreward =| itemreward = | qreward = | insta =| instachefe = }}


Campos Obrigatórios

Campo Descrição
levelreq Nível de Base necessário. Se não houver, escreva "Nenhum".

Campos Opcionais

Campo Descrição
joblevelreq Nível de Classe necessário.
classreq Classe que o jogador deve ser para completar a Quest. Ex.: Apenas Cavaleiros podem completar a Quest de Avanço Ofensivo
skillreq Habilidade necessária que o jogador precisa ter para completar a Quest.
partyreq Número de membros de um grupo para completar a Quest.
itemreq Itens requeridos que a Quest pede.
zenyreq Quantidade de Zeny que é requerida pela Quest.
hunting Lista os monstros que deverão ser mortos para completar a Quest.
questreq Pré-requisitos que o jogador deve fazer antes de começar a Quest.
baseexpreward Recompensa em EXP de Base.
jobexpreward Recompensa em EXP de Classe.
itemreward Itens que o jogador ganha como recompensa.
qreward Outras recompensas da quest, como Acesso à Calabouços e etc...
insta Coloque "yes" para sinalizar que a quest é uma instância.
instachefe Coloque o MVP final da instância. (utilize a predefinição monster).
