Notas

  • Confira os materiais necessários para a produção de poções na página Alquimia.
  • Para produção de cada poção, é preciso uma quantidade de determinados itens.
  • Os Manuais de Criação precisam estar no inventário.
  • Quando a habilidade for utilizada aparecerá uma lista de itens que podem ser fabricados, dependendo dos manuais de criação no inventário.
