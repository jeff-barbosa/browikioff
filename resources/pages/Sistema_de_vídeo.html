Suas aventuras em Ragnarök Online poderão ser vistas por outros jogadores através da ferramenta de gravação. Os vídeos não podem ser visualizados em outros programas, como o Windows Media Player, porque possuem o formato .rtf.

Para gravar um vídeo, basta clicar no botão “REC” na Janela de Informações para exibir a ferramenta de vídeo. Para assistir uma gravação, abra o Patch Client e selecione "Replay".



Gravar vídeos por esse sistema não causará lentidão (lag) porque o sistema utiliza as informações do servidor para salvá-lo, por isso, os arquivos de vídeo são bem leves: um vídeo de 4 minutos possui, aproximadamente, 1 MB.

Além disso, os vídeos simulam o jogo inteiro, ou seja, ao assistir um replay, você poderá movimentar a câmera, abrir a janela de equipamentos e etc. Usuários que assistem vídeos gravados por você poderão ver a descrição dos itens e até arrastá-los para fora da janela com o intuito de jogá-lo no chão, mas como é apenas um vídeo, nada acontecerá.

Iniciar Clique para iniciar a gravação do seu vídeo. Você precisa nomear sua gravação primeiro.
Pausar Pausa o vídeo e salva na pasta do jogo. Não é possível retomar a gravação no mesmo arquivo.
Fim Fecha a janela de gravação e encerra o uso do sistema.
Opções Clique para definir quais informações do seu personagem serão compartilhadas com quem for assistir ao vídeo.
Basic Info Habilita a visualização das Informações Básicas do personagem.
All Libera a visualização de tudo: habilidades, itens, equipamentos e etc.
Equip Ativa a visualização dos equipamentos do personagem.
Item Permite que o espectador veja seu inventário.
Skill Disponibiliza a visualização da Árvore de Habilidades.
Party Permite visualizar membros do grupo.
Chatting Exibe seu chat e conversas particulares.
Shotcut Ativa a visualização dos atalhos de habilidades.
User defined file name Inicia uma gravação sem necessidade de nomear o vídeo.
Repeated file check Avisa se há um arquivo com nome igual.
Assistindo o vídeo
Ícone Nome Descrição
Início Inicia, pausa ou retoma ao vídeo gravado.
